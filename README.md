docker > redmine + mysql
================================================================================
- 레드마인 사용성 체험을 위한 저장소임
- [`docker`](https://docs.docker.com/engine/installation) 및 [`docker-compose`](https://docs.docker.com/compose/install)가 설치되어 있는 환경에서 바로 실행가능

run
--------------------------------------------------------------------------------

first
```bash
$ docker-compose up
```

next
```bash
$ docker-compose start
...
$ docker-compose stop
```
